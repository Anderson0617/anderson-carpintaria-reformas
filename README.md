## 📌 Sobre o projeto

Este projeto foi desenvolvido para apresentar de forma profissional os serviços da **Anderson Carpintaria e Reformas**, com visual moderno, responsivo e voltado para captação de clientes.

O site conta com:

- 🖼️ Galeria de imagens por categoria
- 🎥 Vídeo principal
- ⭐ Avaliações de clientes
- 🔐 Painel administrativo
- 🌍 Conteúdo global via Supabase
- 🚀 Publicação via GitHub Pages
- ☁️ Worker Cloudflare para automações de publicação
- 📊 Contador global de visitas
- 📍 Registro aproximado de região, cidade e estado dos acessos
- 🧰 Fluxo duplo de publicação: Supabase e GitHub
