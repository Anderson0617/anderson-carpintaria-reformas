# Anderson Carpintaria e Reformas

![Anderson Carpintaria e Reformas](./73-ChatGPT-favicom-removebg-preview.png)

Site profissional de apresentação de serviços em carpintaria, estruturas em madeira, telhados, decks, pergolados, piso vinílico, alvenaria e reformas.

## Serviços

- Carpintaria
- Estruturas em madeira
- Decks
- Pergolados
- Telhados e coberturas
- Portas e aberturas
- Piso vinílico click
- Alvenaria
- Reformas
- Revestimentos
- Lajes e vigamento
- Caixaria e radier

## Sobre o projeto

Este projeto foi desenvolvido para apresentar de forma profissional os serviços de Anderson Carpintaria e Reformas, com visual moderno, galeria de imagens, vídeo principal, avaliações e painel administrativo.
